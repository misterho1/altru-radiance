# How to Import

1. **Shared list (recommended for buckets 1-14):** In Google Ads Editor, go to *Keywords and targeting → Negative keywords, campaign → Shared library → Negative keyword lists*. Create a list named "Altru — Universal Negatives," paste the 14 buckets above (one keyword per line), then attach the list to all three campaigns. This way you maintain one source of truth.
2. **Campaign-level (recommended for the Campaign-Specific Negatives sections):** Select the target campaign in Editor, open *Keywords, Negative → Campaign*, and paste the campaign-specific block. Editor will default to broad match, which is what you want — leave match type alone.
3. **Tip:** After pasting, click *Post* in Editor to push to Google Ads. Re-run a search terms report 7-14 days later and add anything new the buckets missed into the shared list rather than per-campaign — easier to maintain.

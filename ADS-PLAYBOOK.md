## 4. Square Appointments Conversion Tracking

### Step 1 — Create thank-you pages on altruradiance.com

Create three new pages with these URLs:
- `/thanks-restorative.html`
- `/thanks-soothe.html`
- `/thanks-procell.html`

Each page should:
- Confirm the booking ("Thank you — your session is reserved")
- Include nav back to home, journal link, and contact link
- Include `<meta name="robots" content="noindex">` in `<head>`
- Load fast (under 1s)
- Include the Google Ads conversion tag for that specific service

### Step 2 — Configure Square redirects

Square Dashboard → Online Booking → Settings → After Booking:
- Set "Redirect to custom URL after booking" per service:
  - Restorative + Buccal Massage → `https://altruradiance.com/thanks-restorative`
  - Soothe Lymphatic Drainage → `https://altruradiance.com/thanks-soothe`
  - Procell Microchanneling → `https://altruradiance.com/thanks-procell`

**Fallback if Square doesn't allow per-service redirects on your plan:**
- Single redirect: `https://altruradiance.com/thanks?service={service_name}`
- Parse `service` query param on a single `/thanks` page and fire the matching conversion tag in JavaScript

### Step 3 — Create three conversion actions in Google Ads

Google Ads → Tools → Conversions → New Conversion Action → Website:

| Name | Category | Value | Count | Window |
|---|---|---|---|---|
| Booking – Restorative | Purchase | $250 | One per click | 30 days |
| Booking – Soothe | Purchase | $250 | One per click | 30 days |
| Booking – Procell | Purchase | $300 | One per click | 30 days |

Mark all three as **primary** conversions. Don't track form submissions or page views as primary — those are signals, not the goal.

### Step 4 — Install conversion tags

Google generates a `gtag` snippet per conversion. Install one per thank-you page (in `<head>`).

Global site tag (only one per site, in every page `<head>`):
```html
<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=AW-XXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'AW-XXXXXXXXX');
</script>
```

Per thank-you page, the event snippet (one example, Restorative):
```html
<script>
  gtag('event', 'conversion', {
    'send_to': 'AW-XXXXXXXXX/AbC-D_efG-hIjKlMnO',
    'value': 250.0,
    'currency': 'USD',
    'transaction_id': ''
  });
</script>
```

### Step 5 — Verify

Install Google Tag Assistant Chrome extension. Walk through one test booking per service. Confirm:
- Global tag fires on every page
- Conversion tag fires only on the matching thank-you page
- No duplicate firing
- Correct conversion ID and label match what's in Google Ads

### Step 6 — Wait, then optimize

- First 30 days: don't touch bid strategy. Let Maximize Conversions learn.
- Once you have 30+ conversions, switch to Target CPA at $40 (or your actual cost-per-booking goal).
- After 60 days, prune negatives based on Search Terms Report. Add 5–10 negatives weekly until search-terms are 90%+ relevant.

## 8. Things That Will Tank Performance — Avoid

- Running Display or Search Partners on day 1 (turn these OFF on every Search campaign)
- Bidding on "TMJ" or "jaw pain" keywords (wrong buyer — bad reviews follow)
- Promising "20% off first visit" in ad copy (trains discount shoppers, attracts wrong client)
- Sending ads to homepage instead of service-specific landing pages
- Skipping conversion tracking on day 1 (every later decision becomes guesswork)
- Targeting "Interest" locations instead of "Presence" (you'll get clicks from people physically not in SLC)
- Letting Search Terms Report go unchecked for 30+ days (negatives compound)
- Bidding on Hydrafacial or competitor names without honest "alternative" framing (Quality Score penalty)

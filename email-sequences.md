# Altru Radiance — Post-Visit Email Sequences

These are the automated email flows sent from Square (or a paired ESP) following each of the three core services at Altru Radiance. Each sequence runs four emails: a booking confirmation, a 24-hour aftercare note, a one-week check-in, and a five-week rebook prompt framed around the regenerative arc rather than a calendar reminder. All copy is written in Mechelle's first-person voice. No emojis, no exclamation points, short lines, no fluff.

## Implementation notes

- Square's built-in email automation supports a booking confirmation and a single post-visit follow-up, but does not natively support a three-step drip at 24-hour, 1-week, and 5-week intervals per service. Plan to pair Square with a dedicated ESP for the drip.
- Recommended stack: keep Email 1 (confirmation) in Square so it triggers directly off the booking event, and run Emails 2 through 4 from Mailchimp, ConvertKit, or Klaviyo. Klaviyo is the strongest fit if Mechelle wants per-service segmentation and revenue attribution on rebook clicks.
- Sync Square appointments to the ESP via Zapier or a native Square integration. Trigger each post-visit sequence on "Appointment Completed" filtered by service name, so each of the three flows fires only for its matching service.
- Use the "Appointment Completed" timestamp as the anchor for delays — 24 hours, 7 days, and 35 days — rather than fixed send times, so the cadence stays aligned to each client's actual visit.
- Send from mechelle@altruradiance.com with a reply-to that routes to her inbox. Authenticate the sending domain (SPF, DKIM, DMARC) before turning the flows on so the slower-cadence emails do not land in Promotions.
